WebDisk Installation
====================

1. Entpacke alle Dateien auf deinen Webserver (z.B. in /WebDisk)
2. Rufe die URL im Browser auf (z.B. https://www.tubox.de/WebDisk/install.php)
3. Gib die Zugangsdaten deiner MySQL-Datenbank ein und starte die Installation
4. Nach erfolgreicher Installation wird eine config.php angelegt und du kannst WebDisk normal verwenden

Wenn du die Installation wiederholen willst, lösche einfach die config.php und rufe install.php erneut auf.

Beispiel für eine MySQL-Tabelle findest du in webdisk.sql (optional)

Sicherheitshinweis: config.php enthält sensible Zugangsdaten – stelle sicher, dass sie nicht öffentlich zugänglich ist!
